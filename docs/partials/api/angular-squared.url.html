url
service in module angular-squared

Description

Provides a simple filter to apply any url rewrites that have been configured. intended to be used wherever you have a url in your html. Recommended even if you are currently not doing any modifications to the url, as it will allow you to make this change in the future with minimal effort.

Dependencies

  • $urlRewrite

Usage

In HTML Template Binding

{{ url_expression | url }}

In JavaScript

$filter('url')(url)

Parameters

ParamTypeDetails
urlstring

url to be transformed

Returns

string

transformed url
