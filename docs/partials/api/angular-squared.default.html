default
service in module angular-squared

Description

Provide a default string when value is null or empty

Usage

In HTML Template Binding

{{ default_expression | default:default }}

In JavaScript

$filter('default')(expression, default)

Parameters

ParamTypeDetails
expressionstring

Expression to be evaluated

defaultstring

Value that will be displayed if the input is null or undefined

Returns

string

default value if expression evaluates to null or undefined
