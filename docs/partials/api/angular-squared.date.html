date
service in module angular-squared

Description

Wraps angular's built-in date filter, and calls the configured date parser before formatting.

This allows you to maintain the same html structure, but easily bind to models retrieved from the server side ajax as is.

Provides the exact same functionality as existing angular dateFilter with the addition of being able to pre-parse the date.

Refer to angular documentation for usage: Angular documentation: date

Dependencies

  • $dateParser

Usage

In HTML Template Binding

{{ date_expression | date }}

In JavaScript

$filter('date')(input)

Parameters

ParamTypeDetails
inputstringnumberdate

date in whatever format your $dateParser.parse is configured to handle

Returns

string

formatted date string
