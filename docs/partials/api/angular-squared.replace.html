replace
service in module angular-squared

Description

Provides an easy filter to perform string replacements inline in an angular template.

Usage

In HTML Template Binding

{{ replace_expression | replace:searchString:replaceString }}

In JavaScript

$filter('replace')(expression, searchString, replaceString)

Parameters

ParamTypeDetails
expressionstring

Input string to perform replacements on

searchStringstring

Substring that will be replaced

replaceStringstring

Replacement value

Returns

string

Processed result after substitutions
