$dateMapper
service in module angular-squared

Description

In many cases, when interacting with different server side backends, each has their own unique way of formatting dates when serializing to JSON, due to JSON not natively supporting date objects.

This can lead to some rather ugly hacks both client and server side to translate between them. This service allows you to configure a mapping function when you setup your app, and will automatically be utilized within the framework for interpreting server side dates as javascript Date() objects.
